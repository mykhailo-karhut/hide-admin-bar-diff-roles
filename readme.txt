The admin panel may be hidden for the user's specific role.

== Description ==

Appropriate for sites where there is a few users.
With this plugin you can hide admin panel for depending on the role user. 

== Installation ==

1. Upload the `hide-admin-bar-for-different-roles` directory to the `/wp-content/plugins/` directory of your site.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Use the 'Tools => Admin bar'.
